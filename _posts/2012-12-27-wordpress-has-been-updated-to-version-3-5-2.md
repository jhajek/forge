<h2><a href="https:&#47;&#47;codex.wordpress.org&#47;Version_3.5" target="_blank">https:&#47;&#47;codex.wordpress.org&#47;Version_3.5 <&#47;a><&#47;h2></p>
<h2>Highlights<&#47;h2></p>
<ul>
<li><b>New Media Manager<&#47;b>
<ul>
<li>Beautiful interface: A streamlined, all-new experience<&#47;li>
<li>Create galleries faster with drag-and-drop reordering, inline caption editing, and simplified controls<&#47;li>
<li>Insert multiple images at once with Shift&#47;Ctrl+click<&#47;li><br />
<&#47;ul><br />
<&#47;li><br />
<&#47;ul></p>
<ul>
<li><b>New Default Theme - Twenty Twelve<&#47;b>
<ul>
<li>Simple, flexible, elegant<&#47;li>
<li>Mobile-first, responsive design<&#47;li>
<li>Gorgeous Open Sans typeface<&#47;li>
<li>Uses the latest Theme Features<&#47;li><br />
<&#47;ul><br />
<&#47;li><br />
<&#47;ul></p>
<ul>
<li><b>Admin Enhancements<&#47;b>
<ul>
<li>New Welcome Screen<&#47;li>
<li>Retina-Ready (HiDPI) Admin<&#47;li>
<li>Hide Link Manager for new installs<&#47;li>
<li>Better accessibility for screenreaders, touch devices, and keyboard users<&#47;li>
<li>More polish on admin screens, including a new color picker<&#47;li><br />
<&#47;ul><br />
<&#47;li><br />
<&#47;ul></p>
<ul>
<li><b>For Developers<&#47;b>
<ul>
<ul>
<li>WP_Comment_Query and WP_User_Query accept now meta queries just like WP_Query<&#47;li>
<li>Meta queries now support querying for objects without a particular meta key<&#47;li>
<li>Post objects are now instances of a WP_Post class, which improves performance and caching<&#47;li>
<li>Multisite's switch_to_blog() is now significantly faster and more reliable<&#47;li>
<li>WordPress has added the Underscore and Backbone JavaScript libraries<&#47;li>
<li>TinyMCE, jQuery, jQuery UI, and SimplePie have all been updated to the latest versions<&#47;li>
<li>Image Editing API for cropping, scaling, etc., that uses ImageMagick as well as GD<&#47;li>
<li>XML-RPC: Now always enabled and supports fetching users, managing post revisions, searching<&#47;li><br />
<&#47;ul><br />
<&#47;ul><br />

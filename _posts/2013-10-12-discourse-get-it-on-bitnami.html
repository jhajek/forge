After you download bitnami package:

  • Create a virtual machine on virtual box </li>
  • Choose existing hard disk </li>
  • Select the main vmdk(the one without s001.vmdk, s002.vmdk etc...)</li>
  • Set network configuration as bridged adapter</li>
  • Start the new vm you just made</li>
  • Log in, I believe user name and pass is bitnami then you are asked to change your password</li>
  • There will be an IP address displayed; put that into your browser and discourse will start up!</li>
